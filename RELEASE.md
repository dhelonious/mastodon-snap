# Unofficial Snap for Mastodon 4.5.7

See the release notes for Mastodon: https://github.com/mastodon/mastodon/releases/tag/v4.5.7

This release also includes:

* ffmpeg 8.0.1
* libvips 8.18.0
* nginx 1.29.5
* node 20.20.0
* pgbouncer 1.25.1
* postgres 18.2
* redis 8.6.1
* ruby 3.4.7
