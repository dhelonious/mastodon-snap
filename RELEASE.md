# Unofficial Snap for Mastodon 4.6.0

See the release notes for Mastodon: https://github.com/mastodon/mastodon/releases/tag/v4.6.0

This release also includes:

* ffmpeg 8.1.2
* libvips 8.18.3
* nginx 1.31.2
* node 24.17.0
* pgbouncer 1.25.2
* postgres 18.4
* redis 8.8.0
* ruby 4.0.5
